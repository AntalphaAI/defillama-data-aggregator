## 🆘 常见问题

### Q1: DefiLlama为什么是免费的？

**A:** DefiLlama是一个开源项目，提供免费的DeFi数据访问，不需要API密钥。

### Q2: 如何获取免费API密钥？

**A:** 访问各平台官网注册免费账户：
- CoinMarketCap: https://pro.coinmarketcap.com/signup
- Coinglass: https://coinglass.com/pricing
- Nansen: https://nansen.ai/pricing

### Q3: 配置后如何测试？

**A:** 运行测试命令：
```bash
crypto-data test --all
```

### Q4: 如何清理缓存？

**A:** 运行清理命令：
```bash
crypto-data cache clear --all
```

### Q5: 如何选择输出格式？

**A:** 使用format参数：
```bash
--format json    # JSON格式
--format table   # 表格格式
--format csv     # CSV格式
--format pretty  # 彩色输出
```
