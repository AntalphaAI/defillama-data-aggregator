## 🎯 功能清单

### 核心命令（7个）

```bash
defillama-data --version          # 查看版本
defillama-data defillama tvl      # 全网 TVL
defillama-data defillama protocol -n <name>  # 协议 TVL
defillama-data defillama protocols  # 协议列表
defillama-data defillama chain -n <name>    # 链 TVL
defillama-data defillama yields    # 收益池
defillama-data health             # 健康检查
defillama-data status             # 系统状态
```

### 输出格式（4种）

```bash
-f pretty   # 美化输出（默认）
-f table    # 表格格式
-f json     # JSON 格式
-f csv      # CSV 格式
```
